CoCreate-position

A draggable, customizable Pop up that toggle hiddent content via attributes. Great for multitasking and showing multiple views.

Usage

CoCreate-position uses <a> tag.

When you click <a> tag, position comes to the surface. 

                        
                            <button>
                              <a target="position" href="https://server.cocreate.app/CoCreate-adminUI/module_datatable.html"
                                position-ajax="false"
                                position-width="520px"
                                position-height="100%"
                                position-color="#229954"
                                position-header="false"
                                position-x="0px"
                                position-y="0px"
                                data-open_in="page">
                                Show position On Page
                              </a>
                            </button>

Attributes

  • target string required

    Target must be "position".

  • href string required

    Href attribute indicates html file to set in position.

  • position-ajax boolean optional

    Attribute whether to use ajax or not.

  • position-width string optional

    Width of position. Unit can be pixel or percent.

  • position-height string optional

    Height of position. Unit can be pixel or percent.

  • position-color color optional

    Color of position border.

  • position-header boolean optional

    Attribute whether to show header or not.

  • position-x string optional

    Position X of position.

  • position-y string optional

    Position Y of position.

  • open_in string optional

    Attribute where to show position in. "page" for current page. "parent" for parent page. "root" for root page.
